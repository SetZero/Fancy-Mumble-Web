About

The Fancy Mumble Web is a free and open-source VoIP web client. You are free to use, share and modify it to your needs. You can even run it on your own Server for maximum privacy. Unlike Discord, TeamSpeak, Skype, etc. Fancy Mumble Web can be modified and improved to your needs without any legal concerns. Also, if you ever have the need for any special feature you are free to ask about our GitHub page for it.

Usage

The basic usage of Fancy Mumble is quite simple: just pick a Username and click login. You will automatically be connected to the default server which lets you talk to strangers on the Internet. If you want to choose your own Server just click on the "Advanced" button below the input. You can then choose the Server you want to connect to.
